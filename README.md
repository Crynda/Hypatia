# Hypatia

Hypatia es una aplicación de escritorio desarrollada en JavaFX para facilitar el aprendizaje y la resolución de problemas de estadística mediante una interfaz moderna e intuitiva.

Este proyecto está pensado como una herramienta **Open Source** de uso libre, con el propósito de apoyar a la 
comunidad estudiantil, docentes y cualquier persona que necesite realizar cálculos estadísticos. 
Su objetivo es ofrecer una experiencia sencilla, intuitiva y accesible, permitiendo comprender y resolver 
problemas estadísticos sin necesidad de contar con conocimientos avanzados en la materia.
